# challenge-frontend-ego

### Prerequisites

- npm

  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/valentinaless07/challenge-frontend-ego
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the project and enjoy!
   ```sh
   npm run dev
